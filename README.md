## 🚀 Installation

### Quick Start (5 minutes)

1. **Download** the installer from the [releases page](https://github.com/c9infotech/Bill-Sency/releases)

2. **Run** `BillSency-Setup-1.0.0-beta.exe`
   - Windows may show a security warning (click "Yes" - this is expected for beta versions)

3. **Accept** the license agreement

4. **Choose** installation options
   - Default location: `C:\Program Files\Bill-Sency`
   - Optional: Create desktop icon

5. **Launch** Bill-Sency

6. **First-time setup:**
   - App starts automatically with local database
   - Login with credentials provided after installation
   - **Important:** Change default password immediately after first login

7. **Install demo apps** (optional)
   - POS System
   - Inventory Management
   - And more...

📖 **Detailed installation guide:** [INSTALLATION.md](INSTALLATION.md)
